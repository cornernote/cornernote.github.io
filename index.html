Yii Extensions

  • Yii Audit Module - Track and display usage information including page requests, database field changes, php errors and yii logs.
  • Yii Menu Module - Interface to manage menus and code-generation to deploy them into your application.
  • Yii Embed for OpenCart - A lightweight Yii application embedded into OpenCart.
  • Yii Email Module - Email system with templates and email queuing.
  • Yii Token Manager - Manages tokens that can be validated, used and expired.
  • Yii Return Url - Maintain the state of Return Urls by using the request.

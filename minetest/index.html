Mods

Armor

Craft and attach armor to your inventory.

Craft Guide

Shows crafts for all loaded mods.

Infinite Chest

Chests with slots that never run out.

Particles

Generates particles in the world when you dig blocks, smoke, etc.

Inventory Plus

Allows additional formspec buttons to be added to the player inventory.

Bags

Attach bags to your inventory to increase item storage capacity.

Home GUI

Set home position and then go to it using the inventory screen.

WorldEdit GUI

Provides a player interface for worldedit mod.

Deploy Nodes

Craftable nodes that when placed turn into shapes or structures.

Bookmarks GUI

Players can set bookmark positions and then go to them using the inventory screen.

Towntest

Grow a town in minetest.

Node Owner

Blocks you place are owned by you and others cannot dig them or place onto them.

Hill Nodebox

Craft and place hill-like nodes.

Critters

Adds heaps of critters that run around in the game.

No Mapgen

Generates an empty or flat world.
